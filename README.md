# EncuestasApp

Aplicación web estática con:

- Inicio de sesión (demo local): `admin / 123456`.
- Registro manual de respuestas tipo encuesta.
- Cargue masivo de datos desde archivos `.json` o `.csv`.
- Resumen con métricas y tabla de resultados.

## Ejecutar

Abre `index.html` en el navegador o usa un servidor estático:

```bash
python3 -m http.server 8000
```

Luego visita `http://localhost:8000`.
